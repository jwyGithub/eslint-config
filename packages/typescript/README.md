### 安装

```sh
pnpm add @jiangweiye/eslint-config-ts -D
or
yarn add @jiangweiye/eslint-config-ts -D
or
npm install @jiangweiye/eslint-config-ts -D
```

### 使用

```js
module.exports = {
    extends: '@jiangweiye/eslint-config-ts'
};
```
